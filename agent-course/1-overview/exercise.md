# 第 1 课作业：先看懂这个极简 AI Agent 在做什么

## 作业目标

通过拆解当前项目的最小闭环，建立对整个 AI Agent 教程项目的全局认知。

## 作业 1：画出最小架构图

请你用自己的方式画出下面这张图，不要求工具，纸笔也可以：

`CLI -> Agent -> Tools -> UI`

并在每个模块下面写一句话说明职责。

### 提交要求

- 图中至少标出 4 个核心模块
- 每个模块至少写 1 句职责说明
- 尝试标出数据或控制流方向

### 完成标准

你能够不看源码，大致说清一条用户请求是如何在系统里流动的。

## 作业 2：用自己的话解释“极简版”

不要照抄文档，试着用 100 到 200 字说明：

- 当前项目已经具备了什么
- 当前项目还缺少什么
- 为什么它适合作为教程起点

### 提交要求

- 使用自己的语言
- 同时写出“已有能力”和“未完成部分”
- 不要只写功能清单，要有判断

### 完成标准

你能够解释为什么这个项目既已经可运行，又仍然处在教学友好的最小实现阶段。

## 自查问题

在提交前，确认自己能回答：

- 入口文件是哪一个？
- Agent 的装配中心是哪一个？
- 哪几类工具构成了当前能力边界？
- UI 为什么不是可有可无的附属层？
